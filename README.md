# IoT Development Trainer Kit PCB Design

## Overview

This project presents a modular IoT Development Trainer Kit PCB designed using Cadence Allegro PCB Designer.

The objective of this PCB is to provide a reusable development platform where different IoT sensors and modules can be connected externally using jumper wires for rapid prototyping and embedded system learning.

Instead of designing a new PCB for every project, users can simply connect the required modules and develop multiple IoT applications.
